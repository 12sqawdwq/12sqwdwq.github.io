# Architecture Module Study Document

## Module Goal

Build a closed learning loop for Architecture: concepts, formulas, engineering applications, and verifiable artifacts. This module covers: Boolean algebra, Logic gates, Sequential logic, FSM, Pipeline, Cache, ISA, GPU.

## Recommended Resource

[MIT 6.004 Computation Structures](https://ocw.mit.edu/courses/6-004-computation-structures-spring-2009/)

## Foundational Source

[von Neumann, 1945, First Draft of a Report on the EDVAC](https://archive.org/details/firstdraftofrepo00vonn)

## Core Formula / Model

Performance equation: `CPU Time = Instruction Count * CPI * Clock Cycle Time`. Optimize only after locating the bottleneck.

How to study the formula:

- Identify every variable and decide whether it is physical, geometric, statistical, or computational.
- Write the assumptions that make the formula valid.
- Verify one minimal example using numbers, a plot, or code.
