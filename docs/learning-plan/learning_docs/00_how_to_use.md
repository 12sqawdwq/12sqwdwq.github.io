# How to Use This Learning Pack

This learning pack is generated from 270_day_engineering_cs_chip_plan.xlsx, starting on 2026-05-10. The main file is one large 270-day table. Each day includes the topic, key question, practice task, required formula/model, online resources, and a local module document.

Suggested daily rhythm:

1. Read the day's Key Question first and write your intuitive answer in five minutes.
2. Open the Online Resource and Local Module Doc, but stay focused on the day's topic.
3. Complete the Daily Deliverable. Leave at least one artifact: code, plot, derivation, note, or design sketch.
4. Use the Daily Prompt template to reflect on the real problem, core equation, failure conditions, cross-domain links, and what you truly understood.

Suggested time blocks:

- 30 minutes: reading and formula understanding.
- 60 minutes: derivation, simulation, or coding.
- 20 minutes: package the day's deliverable.
- 10 minutes: reflection and tomorrow's gap list.
