# Networks Module Study Document

## Module Goal

Build a closed learning loop for Networks: concepts, formulas, engineering applications, and verifiable artifacts. This module covers: TCP/IP, Routing, DNS, TLS.

## Recommended Resource

[Stanford CS144](https://www.scs.stanford.edu/10au-cs144/)

## Foundational Source

[Cerf and Kahn, 1974, A Protocol for Packet Network Intercommunication](https://doi.org/10.1109/TCOM.1974.1092259)

## Core Formula / Model

Throughput estimate: `Throughput ~= Window/RTT`. Window size and round-trip time shape transfer rate.

How to study the formula:

- Identify every variable and decide whether it is physical, geometric, statistical, or computational.
- Write the assumptions that make the formula valid.
- Verify one minimal example using numbers, a plot, or code.
