# Analog IC Module Study Document

## Module Goal

Build a closed learning loop for Analog IC: concepts, formulas, engineering applications, and verifiable artifacts. This module covers: Differential pair, Operational amplifier, PLL, ADC, DAC.

## Recommended Resource

## Foundational Source

## Core Formula / Model

Small-signal gain: `A_v ~= -g_m R_D`. Transconductance turns input voltage variation into output current variation.

How to study the formula:

- Identify every variable and decide whether it is physical, geometric, statistical, or computational.
- Write the assumptions that make the formula valid.
- Verify one minimal example using numbers, a plot, or code.
