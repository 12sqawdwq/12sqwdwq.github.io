# Algorithms Module Study Document

## Module Goal

Build a closed learning loop for Algorithms: concepts, formulas, engineering applications, and verifiable artifacts. This module covers: Complexity, Recursion, Dynamic programming, Graph theory, Hash tables, Compilers.

## Recommended Resource

[MIT 6.006 Algorithms](https://ocw.mit.edu/courses/6-006-introduction-to-algorithms-spring-2020/)

## Foundational Source

[Dijkstra, 1959, A Note on Two Problems in Connexion with Graphs](https://doi.org/10.1007/BF01386390)

## Core Formula / Model

Complexity: `T(n)` tracks growth with input size. Estimate the order of growth before coding.

How to study the formula:

- Identify every variable and decide whether it is physical, geometric, statistical, or computational.
- Write the assumptions that make the formula valid.
- Verify one minimal example using numbers, a plot, or code.
