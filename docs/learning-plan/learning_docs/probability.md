# Probability Module Study Document

## Module Goal

Build a closed learning loop for Probability: concepts, formulas, engineering applications, and verifiable artifacts. This module covers: Random variables, Gaussian distribution, Central limit theorem, Thermal noise.

## Recommended Resource

[MIT 6.041 Probabilistic Systems Analysis](https://ocw.mit.edu/courses/6-041sc-probabilistic-systems-analysis-and-applied-probability-fall-2013/)

## Foundational Source

[Kolmogorov, 1933, Foundations of Probability](https://archive.org/details/kolmogorov-foundations-of-the-theory-of-probability)

## Core Formula / Model

Mean and variance: `E[X]=sum x p(x)`, `Var(X)=E[(X-mu)^2]`. Noise becomes measurable through statistics.

How to study the formula:

- Identify every variable and decide whether it is physical, geometric, statistical, or computational.
- Write the assumptions that make the formula valid.
- Verify one minimal example using numbers, a plot, or code.
